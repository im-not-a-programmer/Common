# 壹号平台2.0-皇家壹号 -  管理后台

```
|-src  			//开发
|----assets  		//css，img资源
|----unit.js  		//公用方法
|----routers            //路由配置
|----containers         //业务逻辑页面
|----component          //通用组件
|----http          //网络请求
|----reducer          //状态管理申明
|----actions          //状态管理方法
```

## 启动

``` bash

# 开发环境 
npm start | npm run dev

# 测试版
npm run test

# 正式版
npm run build
```
